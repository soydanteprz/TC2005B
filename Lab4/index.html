¿Qué diferencias y semejanzas hay entre Java y JavaScript? Java crea aplicaciones que se ejecutan en una máquina o explorador virtual, mientras que el código JavaScript sólo se ejecuta en un explorador. El código Java necesita compilación, mientras que el código JavaScript está en todo el texto.
¿Qué métodos tiene el objeto Date? (Menciona al menos 5*) Métodos para establecer otros valores al objeto Date o aplicando los métodos setFullYear(), setMonth(), setDate(), setHours(), setMinutes(), setSeconds(), setMilliseconds, setTime(), según el valor que queramos modificar.
¿Qué métodos tienen los arreglos? (Menciona al menos 5*) Métodos de arreglo JavaScript unshift() – Inserta un elemento al inicio del arreglo. pop() – Remueve un elemento del final del arreglo. shift() – Remueve un elemento del principio del arreglo. slice() – Crea una copia sombra del arreglo.
¿Cómo se declara una variable con alcance local dentro de una función? Puedes declarar una variable con alcance local dentro de una función usando la palabra clave var, let, o const, dependiendo de tus necesidades
¿Qué implicaciones tiene utilizar variables globales dentro de funciones? Tiene varias complicaciones como por ejemplo declarar una variable global puede afectar el uso de una variable local lo que puede conducir a confusion para el compilador y además de no poder reutilizar codigo
