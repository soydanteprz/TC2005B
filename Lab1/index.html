¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE? GET: Obtiene información sobre un recurso en el servidor. Es el método más comúnmente utilizado y es idempotente, lo que significa que puede hacer varias solicitudes GET consecutivas sin afectar el estado del recurso.
HEAD: Es similar a GET, pero solo devuelve la cabecera HTTP sin el cuerpo de la respuesta. Es útil para verificar la existencia de un recurso o obtener información sobre él sin descargar el contenido completo.
POST: Envía datos a un servidor para que procese o almacene información. Por lo general, se utiliza para enviar formularios o cargar archivos.
PUT: Actualiza un recurso existente en el servidor. Si el recurso no existe, lo crea.
PATCH: Actualiza parcialmente un recurso existente en el servidor. Es similar a PUT, pero solo actualiza una parte específica del recurso en lugar de reemplazarlo por completo.
DELETE: Elimina un recurso en el servidor.

Un atributo se considera deprecated cuando ya no se recomienda su uso y es posible que no se admita en futuras versiones de HTML. Esto se debe a que se han introducido alternativas más modernas o más seguras para realizar la misma tarea que ese atributo, o porque ese atributo ha sido reemplazado por una nueva función.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5? HTML5 ofrece una serie de mejoras en comparación con HTML4, como una mayor semántica, una mejor integración de multimedia, formularios más avanzados y un mejor soporte para los dispositivos móviles.
¿Qué componentes de estructura y estilo tiene una tabla? Estructura:
  • La etiqueta table define el inicio y el final de la tabla.
  • La etiqueta tr define una fila en la tabla.
  • La etiqueta th define una celda de encabezado en la tabla.
  • La etiqueta td define una celda de datos en la tabla.
Estilo:
  • La propiedad border-collapse define si las bordes de las celdas deben fusionarse o no.
  • La propiedad border-spacing define el espacio entre las celdas de la tabla.
  • La propiedad width define el ancho de la tabla.
  • La propiedad height define la altura de la tabla.
  • La propiedad text-align define la alineación del texto en la celda.
  • La propiedad vertical-align define la alineación vertical del texto en la celda.
  • La propiedad background-color define el color de fondo de la tabla.
¿Cuáles son los principales controles de una forma HTML5?
  1. input - Define un control de entrada de texto.
  2. textarea - Define un control de entrada de texto de varias líneas.
  3. button - Define un botón de control.
  4. select - Define un control de selección de una lista.
  5. option - Define una opción en una lista.
  6. optgroup - Define un grupo de opciones en una lista.
  7. fieldset - Define un grupo de controles.
  8. label - Define una etiqueta para un control.
  9. form - Define una forma.
